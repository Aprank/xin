# Xin
===

Xin is Single Page Application framework using javascript.

![Xin](https://github.com/reekoheek/xin/raw/master/img/logo.png "SPA Framework")

# Install

```
xpax init
```

to rebuild js and css
```
gulp
```

# Watch
