Directive Referrer

[data-parent-referer="uri"]

Parent Referrer directive use to special cases, because you don't need use Parent Referer to direct you to some uri, you just need to use <a href="#uri">...</a> tag to direct you to some uri.

The special case is when you create layout and the layout will be used by more than one view, you need to use Parent referer or some trick to make it.

Example Layout

<a href="#" class="back">Back</a>, [class="back"] will be the key to make it running.
