Xin

Getting Started

Learn how to easily start using Xin in your every awesome project.

Download

You can select which version you want depending on your preference and expertise over github. To start using Xin, all you have to do is download one of the zip file below.

Download Xin v0.1.1

Install

Bower

You can also get the latest release using bower. This release contains source files as well as the compiled CSS and JavaScript files.

$ bower install xin

Git Clone

You can also get the latest release using git clone. This release contains source files as well as the compiled CSS and JavaScript files.

$ git clone https://github.com/xinix-technology/xin.git

pas

You can also get the latest release using pas. This release contains source files as well as the compiled CSS and JavaScript files.

$ pas install

gulp

To rebuild js and css

$ pas install

HTML Setup

Next you just have to make sure you link the files properly in your webpage. Generally it is wise to import javascript files at the end of the body to reduce page load time. Follow the example below on how to import Materialize into your webpage.

One last thing to note is that you have to import jQuery, Underscore and Backbone before importing xin.js!
